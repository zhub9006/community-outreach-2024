## 📋 Event Checklist

### Pre-Event (1 week before)
- [ ] Confirm venue reservations with each community center
- [ ] Arrange outdoor activity supplies (tables, signage, materials)
- [ ] Prepare printed outreach materials and environmental initiative handouts
- [ ] Coordinate volunteer teams for each district
- [ ] Set up social media announcements for each event date

### Day-of-Event
- [ ] Early setup at venue (indoors + outdoors)
- [ ] Weather check — confirm clear conditions before moving outdoors
- [ ] Registration / sign-in table
- [ ] Environmental initiative presentation (indoor)
- [ ] Outdoor community activities and demonstrations
- [ ] Feedback collection and sign-ups

### Post-Event
- [ ] Collect attendee data and feedback
- [ ] Post event recap to social media
- [ ] Update this planning document with attendance numbers
- [ ] Plan follow-up engagement for each district
